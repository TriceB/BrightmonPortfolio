Website for a Restaurant

This Project is for a restaurant website

This project was to create an online menu for a an Italian Pizza Restaurant.

Objective: Provide more visibility for the restaurant and display the pizza menu to more customers.

Using a single page design, the website shows all menu items in a simple but efficient way.

Menu items can easily be added or updated at anytime using the admin interface.
