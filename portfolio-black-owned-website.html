Naturally Black Owned

Naturally Black Owned is a personal project that I created to showcase 900+ Black Owned Businesses

This website utilizes Airtable as a database to store the businesses displayed on the website.

Python is used to access the Airtable API and pull the business information, then Flask/Jinja is used for the web framework.

Any changes or additions made to the businesses in Airtable are automatically added to website.

Technologies Used: , AIRTABLE API
