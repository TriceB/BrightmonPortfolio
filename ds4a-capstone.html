WNBA vs NBA: Analyzing the Player Salary Gap and Revenue Share by League

This is a Capstone Project completed as a Fellow in the Data Skills 4 All (DS4A) Data Engineering Program by Correlation One.

Problem: How can we better understand what the pay gap between WNBA and NBA looks like?

Developed Front-end Data visualiztion using Dash by Plotly featuring data Extracted and aggregated from multiple sources via webscraping, Transformed using Python, and Loaded to AWS S3 for the Data Late and Redshift as the Data Warehouse to create a clearer picture of the salary gap in an interactive way.

Recognized among 15% of students selected to receive Walmart Endowed Scholar Award for exceptional performance in the program and extraordinary contribution to the DS4A mission
